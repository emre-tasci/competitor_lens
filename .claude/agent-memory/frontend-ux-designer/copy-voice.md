UI copy must read like a marketing copywriter wrote it: benefit-driven, confident, calm, Turkish.

**Rule:** Never frame the product as "istihbarat" (intelligence/espionage) or surveillance. Its purpose is helping the **product team stay up to date with the crypto-exchange industry** — competitors' features, announcements, news, gündem.

**Why:** The user explicitly rejected the "Sektör istihbarat paneli" wording (2026-06-30). They asked to reassign every related word, thinking like a marketing copywriter. "İstihbarat" carries the wrong (spy) connotation.

**How to apply:**
- Lead with benefit/outcome ("güncel kalın", "kaçırmayın", "tek ekranda", "bir bakışta görün"), not surveillance verbs.
- Dashboard headline is now "Sektörün nabzı, tek ekranda". Keep the brand name "Product Terminali" as the eyebrow.
- Prefer "takip edin / izleyin / güncel tutun" over "istihbarat / casusluk".
- This applies to page descriptions, metadata, headings, and any new copy. See also [[ui-patterns]].
